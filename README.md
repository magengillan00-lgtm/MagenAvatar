# 🎭 Magen Avatar - 3D Anime AI Companion

**تطبيق ويب تفاعلي بأفاتار أنمي 3D مع محادثة صوتية ذكية**

## ✨ الميزات الرئيسية

- 🎭 **أفاتار 3D احترافي** - فتاة أنمي جميلة باسم Sakura
- 🎤 **محادثة صوتية كاملة** - تحويل الصوت إلى نص والعكس
- 👄 **مزامنة الشفاه** - حركة الفم متزامنة مع الكلام
- 🧠 **ذكاء اصطناعي محلي** - ردود ذكية بدون API خارجي
- ⚙️ **لوحة تحكم** - تخصيص الإعدادات بسهولة
- 🎨 **تصميم حديث** - واجهة سلسة وجميلة
